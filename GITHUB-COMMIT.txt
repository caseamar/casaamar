release: v2026.07.24.66 — direct Asset Upload Manager

Asset Studio
- Users can drag or select images directly in Asset Studio.
- Optional shared context and season can be supplied before upload.
- New images upload directly to GitHub under images/library.
- Upload automatically triggers GitHub sync, Asset Object creation,
  visual analysis and relation recalculation.
- Filename collisions never overwrite silently.
- Explicit replacement mode preserves the existing asset identity,
  manual metadata and relations while queuing visual reanalysis.

Security
- Upload requires Cloudflare secret GITHUB_TOKEN.
- Token is used only server-side in _worker.js.
- No token is exposed to the browser or repository files.

Testing
- Added TEST-ASSET-UPLOAD-MANAGER.md.
