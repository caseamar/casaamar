release: v2026.07.24.62 — automatic asset import, analysis queue and relabeling

Existing images
- Generated asset-manifest.json from the 16 image paths referenced by the current website.
- asset-library.json is prefilled with those current website images.
- Asset Studio can import any future manifest images not yet registered.

Visual analysis queue
- Added POST /api/asset-analyze using multimodal AI.
- Analyzes scene, objects, mood, season, audiences, campaigns, quality,
  format, suggested uses and improvement notes.
- Processes pending images sequentially with progress and retry state.
- Supports single-image reanalysis and full-library reanalysis.
- Previous AI profiles are retained in analysis history.

Automatic relabeling and relation updates
- Visual reanalysis is triggered by new assets, analysis-version changes
  or a manual request.
- Relation scans are triggered by changes to Knowledge Objects, page
  structure, campaigns or asset labels.
- Added GET /api/platform-signature to detect those changes.
- Only AI suggestions are replaced.
- Manual labels, manual links, current usage and locked placements remain intact.
