fix(page-generator): repair runtime failure

Root cause
- Page generation used loadJsonAsset(), but that function did not exist.
- The existing asset helper is assetJson(env, request, pathname).
- Every page-generation request therefore failed before calling OpenAI.

Fix
- Replaced both invalid calls with assetJson().
- Added endpoint-level try/catch with precise JSON diagnostics.
- Page Studio now displays Worker error details instead of a generic message.
- Page Generator health endpoint reports 10.7-page-generator-runtime-fix.
