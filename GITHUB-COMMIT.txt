release: v2026.07.24.65 — scalable GitHub Asset Sync Engine

GitHub is now the master source for image files.

Sync engine
- Recursively scans all supported images under images/ and subfolders.
- Compares GitHub blob SHA and path with Asset Library.
- Detects:
  - new files
  - moved/renamed files
  - replaced file content
  - deleted/missing files
- Move/rename preserves Asset Object, AI analysis and relations.
- Replacement preserves manual metadata and relations, then queues reanalysis.
- Deletion never removes an Asset Object automatically.
- Missing files still in use or locked become replacement_required.

Upload workflow
- New permanent files go to images/library/.
- Asset Studio provides a direct GitHub folder link.
- Sync review is shown before changes are applied.

Testing
