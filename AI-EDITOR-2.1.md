# AI Editor 2.1

Improvements:
- Explains why each Knowledge Object matches.
- Shows only the meaningful changed content rather than repeating the entire article.
- Suggests related Knowledge Objects.
- Shows which tests should run after approval.
- Saves approved relations and test links into the edited object.
- Keeps the object marked as not tested until targeted tests are run.

Editorial flow:
1. Describe the change.
2. Review recommended object and match reasons.
3. Review compact changes.
4. Review related objects and tests.
5. Approve/edit.
6. Run targeted tests.
